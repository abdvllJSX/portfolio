Netflix Movie Recommender

I worked on a public dataset from Kaggle that consists of about 8000 Netflix movies and series. I made use of the content-based recommendation approach and as such used python's NLTK libraries to analyze each movie's descriptions, casts, etc. I then use the Tfidf vectorizer to vectorize the tokenized words and finally use the cosine similarity to measure distances between features(vectors).

Once a user inputs a movie they enjoyed on Netflix, it recommends 10 other movies similar to it thereby saving unnecessary time that will be wasted on looking for movies to watch.

Press this link to access the GitHub repository of the source code.

No Show Appointment

The No show appointment dataset consists of about 110,527 records of patients in brazil and is focused on whether each patient showed up for their medical appointment. The python analysis focused on determining features useful for the prediction of whether or not a patient will be showing up.
